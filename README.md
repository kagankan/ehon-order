# ehon-order

絵本の注文システム。

## 使用技術

- React
- Next.js
- Tailwind CSS
  - GitHub Copilotのサポートを受けてデザインを作りやすいため。
- Firebase
  - Firebase Data Connect
  - Firebase Authentication

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
